pymdp.utils¶

Functions which have uses throughout the library. In particular, the sup norm \(\|f\|_{\infty} = \sup_{x \in X} |f(x)|\) for bounded functions \(f : X \to \mathbb{R}\) on finite spaces.

Given two functions f,g which operate on the state space of a given MDP, computes

\[\begin{split}\|f-g\| &= \sup_{x \in X} |f(x) - g(x)| \\ &= \max_{x \in X} |f(x) - g(x)|\end{split}\]
pymdp.utils.sup_norm(space, fun)[source]¶
Parameters:
  • space (container class) – A collection of points.
  • fun (space -> float) – A function defined on space.
Return norm:

the value of the supremum norm.
Return type:

float

Computes the supremum norm of the function \(f : X \to \mathbb{R}\) defined by

\[\begin{split}\| f \|_{\infty} &= \sup_{x \in X} | f(x) | \\ &= \max_{x \in X} | f(x) |\end{split}\]

Since \(X\) is assumed finite, the supremum can be replaced with a maximum.
